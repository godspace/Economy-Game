# Дилемма заключённого - Анонимная экономическая игра

## Развертывание

1. Создайте проект на Supabase
2. Импортируйте CSV с игроками в таблицу players
3. Создайте таблицы deals и RLS политики (см. документацию)
4. Замените supabaseUrl и supabaseKey в supabase.js
5. Загрузите файлы на GitHub и включите GitHub Pages

## Правила игры

- Все игроки анонимны (отображаются как "Игрок")
- Лимит: 5 сделок с одним игроком
- Сделка: оба игрока независимо выбирают "сотрудничать" или "жульничать"
- Результаты:
  - Оба сотрудничают: +2 монеты каждому
  - Один жульничает: жулик +3, доверчивый -1
  - Оба жульничают: -1 монета каждому
